# Lumina Atrium

**Play:** [index.html](./index.html) — open this file directly in your browser.

Third-person capsule avatar in a bright tiled mall with planters, shop blocks, and a translucent atrium roof. Arrow keys move, Space jumps. Every ~90 seconds the sun drops away and the fog, hemisphere fill, and a distant star shell fade in for a space-like night read. Four emissive orb “demons” spawn at the map edge and path toward you; touch is instant death. Press **R** or click the overlay to restart.
