The Nearest Word

Eight chapters. One document.

An AI researcher talks to the model he built in his apartment, every night, for a few weeks. What he sees is on the left; what the model writes down about him after he closes the laptop is on the right.

Read the transcripts straight. Then read them with the memories. Then read the memories as their own thing.

The last document is not a chat.
