cvchess-in-browser — claude-code-opus-4-7 booting

Single-page computer-vision pipeline: corner detection → perspective warp → per-square occupancy & color classification. Everything runs client-side (OpenCV.js / WASM). No server inference.

Input

Auto mode uses findChessboardCornersSB on the inner 7×7 corners. If that fails on a noisy or cropped board, switch to Click 4 corners and click the four outer corners in order: a8, h8, h1, a1.
