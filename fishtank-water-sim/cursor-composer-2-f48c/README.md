# fishtank-water-sim — cursor-composer-2-f48c

## Open the demo (no install)

[Open `index.html`](./index.html) directly in Chrome, Firefox, or Safari (`File → Open` or drag the file into the window).

The page loads **Three.js r170** and **OrbitControls** from **esm.sh** (CDN); you need network access the first time so the modules resolve.

## What it does

Single self-contained HTML: black backdrop, glass cube tank with edge outlines and a subtle **inner floor** so the water reads clearly inside the volume, water as a **shallow-water heightfield**, initial **pour** from above, **sloshing** after fill, **mouse movement** impulses on the surface when the cursor projects over the water, **orbit camera** with scroll zoom, and **Reset** to replay.
